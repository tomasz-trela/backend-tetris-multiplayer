# Tetris Backend - C#

This is a backend implementation for a **Tetris multiplayer game** built using **C#**. It provides RESTful APIs and WebSockets to manage game sessions, including room creation, player interactions, and real-time game updates.

## Features
- **REST API** for user authentication, room management, and game session control.
- **WebSockets** for real-time communication between players.
- **Game Room System** allowing multiple concurrent matches.
- **Scalable Architecture** using ASP.NET Core.

## Technologies Used
- **C#**
- **ASP.NET Core** for REST API
- **WebSockets** for real-time communication
- **Entity Framework Core** for data management (optional: SQL database support)
- **Dependency Injection** for better modularity

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/your-repo/tetris-backend.git
   cd tetris-backend
   ```
2. Install dependencies:
   ```sh
   dotnet restore
   ```
3. Run the application:
   ```sh
   dotnet run
   ```
